# Matrix

Provides the Matrix bridge adapter for sh1pt cross-network messaging workflows.

## What it does

- Connects an external communication network to sh1pt.
- Supports bridge workflows for sending or receiving messages where implemented.
- Includes setup guidance for required credentials or provider configuration.
- Can send messages or posts through the connected provider.
